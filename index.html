Technologies Used

  • Amazon Web Services (AWS) - used to host and secure the project.
  • Ubuntu Linux - used as the operating system for the server.
  • Apache Web Server - used to serve the HTML content.
  • HTML - used to create the structure and content of the web page.
  • CSS - used to style the web page.
  • JavaScript - used to add interactivity to the web page.

Steps Taken

  1. Provisioned an EC2 instance on AWS - set up the virtual server.
  2. Installed Ubuntu Linux on the instance - configured the operating system.
  3. Installed Apache Web Server - set up the web server software.
  4. Created and deployed an HTML page - developed and uploaded the web content.
  5. Configured security groups to allow HTTP traffic - ensured the server is accessible over the internet.
